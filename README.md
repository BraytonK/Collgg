# React + Tailwind

Collgg Demo Site

## Getting started

Make sure you have [node.js](https://nodejs.org) LTS installed.

```bash
git clone https://github.com/junobuild/examples
cd react/diary
npm ci
```

## Local development

```
Collgg build:

Use NVM 18 + 

* npm ci
* npm i @junobuild/core  
* npm run dev
* npm i -g @junobuild/cli
* juno login    
* npm run build   
* juno init     
* juno deploy 
```
